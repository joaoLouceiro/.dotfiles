# My dotfiles

This is the third iteration of this repo.

The reasoning behind it is that I started using a new OS (Arch BTW) and I am trying to do stuff a bit more bare-bones. 
It's a pretty simple setup, all-round, but I enjoy it.

I am avoiding stuff like Oh-My-Posh/Oh-My-Zsh, trying to get more out of the native shell, etc.

## What I am using:

- [Neovim](https://neovim.io/)
  - [LazyVim](https://www.lazyvim.org/)
- [Kitty](https://sw.kovidgoyal.net/kitty/)
- [Clangd](https://clangd.llvm.org/)

### Other stuff

- [Arch](https://archlinux.org/)
- Nuphy Air75 v2 (Keyboard)
  - [Nuphy's JSON files](https://nuphy.com/pages/json-files-for-nuphy-keyboards?srsltid=AfmBOoqFDkTyT0oJnKvDcq8Rd5eqP48Rb39aHkMJQIbbjes29xRQMT_d)
